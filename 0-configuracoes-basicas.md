# Instalação, configuração de ambiente R e ferramentas de apoio


## Checklist do que fazer
1. Instalar o **R** no seu computador
2. Instalar uma IDE compatível: **RStudio** (IDE clássica) ou **Positron** (novo produto Posit - recomendo)
3. Criar uma **conta no GitHub**
4. Aplicar para o **GitHub Student Developer Pack** (inclui acesso gratuito ao GitHub Copilot para estudantes)
5. Verificar e adicionar seu **e-mail acadêmico ao GitHub**
6. Conectar o GitHub Copilot na IDE escolhida (RStudio ou Positron)


## 1. Instalar o R
O **R** é a linguagem principal utilizada no curso para análise de dados.

* Acesse a página oficial: [https://cran.r-project.org/](https://cran.r-project.org/)
* Selecione o mirror mais próximo e baixe a versão para o seu sistema operacional (Windows, macOS ou Linux).
* Execute o instalador normalmente.

---
## 2. Instalar uma IDE: RStudio ou Positron

### RStudio
* Faça o download da IDE oficial no site da Posit: [https://posit.co/download/rstudio-desktop/](https://posit.co/download/rstudio-desktop/)
* Escolha a versão **Open Source** (gratuita).
* Instale seguindo os passos do instalador.

### Positron
* Acesse o site oficial do Positron: [https://positron.posit.co/download.html](https://positron.posit.co/download.html)
* Baixe e instale a versão para seu sistema operacional.

A escolha entre RStudio e Positron depende da sua preferência (*mas eu recomendo o Positron, que é a nova IDE da Posit e tem integração nativa com o GitHub Copilot*).

## 3. Criar uma conta no GitHub
Se ainda não tiver uma conta no GitHub:
* Cadastre-se em [https://github.com/](https://github.com/)
* Complete com e-mail, nome de usuário e senha.
  * Recomendo que seja o seu e-mail pesssoal e não o e-mail da PUCSP (que você não terá no futuro)
* Verifique seu e-mail para ativar a conta.

## 4. Solicitar *gratuitamente* o Github Copilot **Pro**

Estudantes universitários podem se inscrever no **GitHub Education**, que oferece acesso gratuito ao **GitHub Copilot Pro**.

* Página oficial do programa: [https://education.github.com/pack](https://education.github.com/pack)
* Clique em **Get your pack** e siga as instruções para se inscrever como estudante.
* Você precisará fornecer o seu **e-mail acadêmico** e preencher o formulário de solicitação.


## 5. Verificar e adicionar o e-mail acadêmico
Para aprovar no Github Education é recomendado:
1. Entrar no GitHub ([https://github.com/](https://github.com/))
2. Ir em **Settings → Emails**
3. Adicionar seu e-mail acadêmico como secundário
4. Verificar esse endereço
5. Caso não se aprove, uma alterntiva é definir o e-mail "@pucsp.br" como principal (enquanto estiver na PUCSP).

Documentação oficial do GitHub sobre como gerenciar e-mails:
[https://docs.github.com/en/account-and-profile/how-tos/email-preferences/changing-your-primary-email-address](https://docs.github.com/en/account-and-profile/how-tos/email-preferences/changing-your-primary-email-address)
